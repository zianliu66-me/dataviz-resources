# Assortment of data visualization resources

Small data visualization-related things that are not 
important enough to be placed anywhere else.

* `pyvenn2.py`: Simple 2-condition Venn Diagram with Python. 
Requires `matplotlib` and `matplotlib_venn`.
* `pyGenomeTrack-example.sh`: example on how to use
pyGenomeTrack. Requires `pygenometracks`.
